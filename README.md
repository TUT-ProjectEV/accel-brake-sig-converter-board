# accel-brake-sig-converter-board

## Overview
- センサの出力信号(電圧)を電流信号に変換
    - 4~20mA

## Requirement
### Development
- Kicad 8.0
### Libraries
- ProjectEV-Kicad-Library
    - v8.0

## Usage
- 

## Features
- 動作電源: 12V
- センサ用電源出力: 5V
- 入力信号電圧: 1~5V
- 出力信号電流: 4~20mA
    - 250Ωで終端
